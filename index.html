HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The meta element defiens metadata such as the character set, description, keywords, author, and viewport.
  • The base element defines the base URL for a webpage.
  • The link element connects an external resource to the HTML document.
  • The anchor element creates links to he same webpage or other webpages.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A border box wraps the content and any padding.
  • The content box is the are where your content is displayed
  • The CSS box model as a whole applies to block boxes and defines parts of a box - margin, border, padding, and content.
  • The three ways to style a webpage using CSS include Inline CSS, Internal CSS style sheets, and External CSS style sheets.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git branch: list your branches. a * will appear next to the currently active branch.
  • git checkout: switch to another branch and check it out into your working directory.
  • git merge [branch]: merge the specified branch's history into the current one.
  • git add [file]: add a file as it looks to your next commit (stage).
